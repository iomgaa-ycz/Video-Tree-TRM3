## Metric 页面渲染指导

Metric 是评估指标定义。读者需要了解：指标含义、基线值、阈值、当前状态。

### 色值
`#2460d8`

### 页面结构

1. **Header**（pb-8）：METRIC Badge(彩色) + 达标状态 Badge → 衬线大标题 → 描述
2. **Separator**
3. **数值概览**（py-8）：纯排版指标区（基线值/当前值/阈值 font-heading 大数字 + 竖线分隔）
4. **可视化条**（py-8）：SVG 水平条，暖色调色板
5. **趋势图**（如有）：SVG 折线图
6. **Separator** → 关联实体区
